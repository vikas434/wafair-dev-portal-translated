Queries

fulfillmentOrderDetails

Description

Returns order information including status for a given supplier's multi-channel fulfillment order.

Response

Returns a FulfillmentOrderDetails

Arguments
Name Description
orderDetailsInput - FulfillmentOrderDetailsInput!

Example

Query
query fulfillmentOrderDetails($orderDetailsInput: FulfillmentOrderDetailsInput!) {
  fulfillmentOrderDetails(orderDetailsInput: $orderDetailsInput) {
    fulfillmentOrder {
      ...FulfillmentOrderFragment
    }
    fulfillmentOrderErrors {
      ...FulfillmentOrderErrorFragment
    }
  }
}
Variables
{"orderDetailsInput": FulfillmentOrderDetailsInput}
Response
{
  "data": {
    "fulfillmentOrderDetails": {
      "fulfillmentOrder": FulfillmentOrder,
      "fulfillmentOrderErrors": [FulfillmentOrderError]
    }
  }
}

FulfillmentOrder

Description

The general information regarding a multi-channel fulfillment order, including its status

Fields
Field Name Description
requestId - String! The identifier returned from a successful createFulfillmentOrder mutation request, used to identify this order
status - String! The status of the fulfillment order.

PENDING: Order request has been successfully received (passing syntactic and semantic validation).
NEW: Order has been successfully acknowledged and created in Wayfairâ€™s CastleGate fulfillment system, but is pending some fulfillment validation checks.
ALLOCATED: The fulfillment order has been validated. All items of the order have been allocated from inventory. If one of the items of an order has not been allocated yet, the order status will remain NEW until all items have been.
PARTIALLY_SHIPPED: Some items in the order have been shipped.
SHIPPED: All items of the order have been shipped.
REJECTED: The order was rejected due to some errors. Error details can be found on the FulfillmentOrderDetails type via the fulfillmentOrderDetails or fulfillmentOrderDetailsList query.
ON_HOLD: The order has been placed on hold.
CANCELLED: The order has been cancelled.
orderDate - DateTime The date and time when the order request was accepted by Wayfair
customerOrderNumber - String The order number displayed to the customer, submitted with the createFulfillmentOrder operation
retailer - FulfillmentOrderRetailer Information regarding the retailer of this fulfillment order
shippingAddress - FulfillmentOrderShippingAddress The destination address submitted with the createFulfillmentOrder operation
fulfillmentOrderItems - [FulfillmentOrderItem!] List of item details included on the fulfillment order.
Example
{
  "requestId": "abc123",
  "status": "abc123",
  "orderDate": "2007-12-03T10:15:30Z",
  "customerOrderNumber": "xyz789",
  "retailer": FulfillmentOrderRetailer,
  "shippingAddress": FulfillmentOrderShippingAddress,
  "fulfillmentOrderItems": [FulfillmentOrderItem]
}
